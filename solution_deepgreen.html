수중 데이터 센터: 냉각의 패러다임 전환

수중 데이터 센터란?

수중 데이터 센터는 영국의 스타트업 '딥 그린'에서 생각해낸 방안으로 데이터 센터를 수영장 아래에 지어서 발열 문제와 막대한 전력 소비 문제를 해결하는 것입니다.

어떻게 작동하나요?

수영장 아래에 있는 데이터 센터 내부에는 물이 아닌 열 흡수율이 높은 오일을 가득 채운 후 이 오일이 기기에서 발생하는 뜨거운 열을 탐지하면 열은 파이트와 열교환기를 거쳐 그대로 수영장 물까지 전달되는 방식입니다. 따라서 오일이 매개체가 되어 데이터 센터의 열로 수영장 물을 따뜻하게 데우는 것입니다.

"서버 열로 수영장을 따뜻하게, 쓸모없는 열에너지를 똑똑하게 재활용합니다."
왜 중요한가요?

데이터 센터는 작동 중에 막대한 열에너지를 발생시켜 이를 식히기 위한 높은 냉각 비용이 들고, 수영장은 물을 따뜻하게 유지하기 위한 난방비가 필요합니다. 그치만 수중 데이터 센터는 데이터 센터에서 발생하는 열을 수영장의 난방에 활용함으로써, 데이터 센터는 냉각 비용을 줄이고 수영장은 난방비를 절감할 수 있기 때문에 중요합니다.
